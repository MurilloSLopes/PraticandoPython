# Praticando Python

Repositório criado para armazenar os exercícios, treinos e aplicações desenvolvidos durante meus estudos de Python no curso da Alura.

## Arquivos

- `app.py`: aplicação desenvolvida durante as aulas do curso.
- `treinos.py`: exercícios, testes e práticas feitos ao longo do aprendizado.

## Objetivo

Praticar conceitos fundamentais de Python, como variáveis, condicionais, laços de repetição, funções e estruturação de código.
